## GitHub credentials 업데이트(mac)
> - keychain access로 자격 증명 업데이트
> - 명령줄을 통한 자격 증명 삭제
> - Git에서 GitHub 자격 증명 캐싱

<br>

Git 사용 시 Personal access token으로 자격 증명 가능

또는 Git Credential Manager와 같은 자격 증명 헬퍼 사용 가능
